Get Started in Modern Campus CMS (formerly Omni CMS or OU Campus)

This site is intended for users of existing sites which use the Business and College templates. For new sites or existing sites which use the Crimson templates, relevant content is more likely to be found on the Modern Campus CMS @ SDSU site.

Modern Campus CMS is used by staff/faculty to create official, public-facing SDSU websites and web pages that are SDSU branded, meet accessibility requirements, and render well for mobile devices.

Modern Campus CMS is supported by IT Web Services and Infrastructure, which provides a number of services to support SDSU's web community.
