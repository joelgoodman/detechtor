    Your Online LMU Application

    You may use this application to apply for admission as a:
    • Undergraduate First-Year Student
    • Transfer Student
    Please visit the following application pages if you are interested in joining LMU for:
    • Graduate or Certificate Programs
    • Pre-College Summer Programs
    • Visiting Undergraduate Programs (Summer Session Only)

    For questions, please contact:
    • First-year Students: admission@lmu.edu
    • Transfer Students: transfer@lmu.edu
    Please do not start a new application if:
    • you are a first-year or transfer applicant and have already submitted or started an application for the same entry term (e.g., Fall 2025);
    • you are a first-year or transfer applicant and have submitted a Common Application or Coalition Application for the same term
       

    Returning users:

    Log in to continue an application.

    First-time users:

    Create an account to start a new application.
