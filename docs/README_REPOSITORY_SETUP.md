# Repository Setup Instructions

This directory contains deTECHtor prepared as a standalone repository. Follow these steps to create the new repository:

## Option 1: Create New Repository on GitHub

1. **Create the repository on GitHub**
   - Go to https://github.com/new
   - Repository name: `detechtor`
   - Description: "SpeedyU's open-source technology detection engine"
   - Visibility: Public (or Private, as needed)
   - **Do NOT** initialize with README, .gitignore, or license (we already have these)

2. **Initialize and push**
   ```bash
   cd detechtor-standalone
   git init
   git add .
   git commit -m "Initial commit: Extract deTECHtor from benchmark-agent"
   git branch -M main
   git remote add origin git@github.com:speedyu/detechtor.git
   git push -u origin main
   ```

## Option 2: Use Existing Directory as New Repo

If you want to keep this directory structure:

```bash
cd detechtor-standalone
git init
git add .
git commit -m "Initial commit: Extract deTECHtor from benchmark-agent"
# Then add remote and push as in Option 1
```

## Post-Setup

After creating the repository:

1. **Install dependencies** (if not already done):
   ```bash
   npm install
   ```

2. **Test the installation**:
   ```bash
   node cli.js --url "https://mit.edu" --verbose
   ```

3. **Update benchmark-agent** to use the new repository (see DETECHTOR_MIGRATION_GUIDE.md)
